#  Sistema de Cadastro de Clientes

Este projeto consiste em um sistema simples de cadastro de clientes desenvolvido em Python com banco de dados SQLite.

##  Funcionalidades

- Cadastro de clientes
- Listagem de clientes
- Exclusão de clientes
- Armazenamento em banco de dados local (SQLite)

##  Tecnologias utilizadas

- Python
- SQLite

##  Objetivo

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação, manipulação de banco de dados e simular um sistema básico de gestão de informações, semelhante a cenários reais utilizados em empresas.

##  Como executar o projeto

1. Certifique-se de ter o Python instalado
2. Faça o download ou clone este repositório
3. Execute o arquivo:

```bash
python main.py
