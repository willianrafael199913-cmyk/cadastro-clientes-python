# Sistema de Cadastro de Clientes

Este é um projeto simples desenvolvido em Python com SQLite, com o objetivo de simular um sistema de cadastro de clientes.

##  Funcionalidades
- Cadastro de clientes
- Listagem de clientes
- Exclusão de clientes

##  Tecnologias utilizadas
- Python
- SQLite

##  Objetivo do projeto
Este projeto foi desenvolvido para praticar lógica de programação, manipulação de banco de dados e organização de informações, simulando um ambiente real de controle de dados.

## ▶️ Como executar
1. Instale o Python
2. Execute o arquivo:

```bash
python main.py
